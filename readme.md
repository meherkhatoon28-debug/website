This page is my personal website that is currently deployed, it allows for people to get some more details and About Me information. It also allows other people to easily access my other online profiles and see my work experience/internship/awards in an interactive way. I made this by first pulling a basic site from the main branch of the WICS web dev repo, then completely changing it, adding more projects, picking a blue floral design and just overall making it my own. I used the React framework, next.js, tailwind CSS for styling and some animations, and deployed it with git and Vercel. I had many bugs, especcially with the local host not connecting and/or running properly, but thankfully I was able to fix it by doing some research and it all worked out!
